# Third Eye for Blind Person

A smart assistive system designed for visually impaired individuals. The project uses sensors to detect obstacles and provides real-time alerts through sound or vibration, helping users navigate safely and independently.

## Features
- Obstacle Detection
- Real-Time Alerts
- Easy to Use
- Low-Cost Solution

## Technologies
- Arduino
- Ultrasonic Sensor
- Buzzer/Vibration Motor
